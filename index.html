About This Server

This private server is maintained by ITCPR to support theoretical and computational physics research, data storage, code development, and remote access for authorized researchers, staff, and collaborators.

Access Policy

Access is strictly limited to authorized personnel. Unauthorized use or access attempts will be logged and reported.

  • Personal credentials only — sharing prohibited.
  • Multi-factor authentication required where applicable.
  • SSH keys must be registered with the system administrator.

Acceptable Use Policy

All users agree to the following conditions:

  • Use resources solely for research and educational purposes.
  • No unauthorized data access, sharing, or illegal activity.
  • Software installations require prior approval.
  • Maintain server stability and respect resource limits.

Data & Security Policy

  • All activities are monitored and logged.
  • Confidential data must be encrypted during transmission and at rest.
  • Security incidents must be reported immediately.
  • Inactive accounts are reviewed regularly.
